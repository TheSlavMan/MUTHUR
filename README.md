# MUTHUR ‚Äî Alien Universe Project for Stellaris

**MUTHUR** is an expansive Stellaris mod that aims to bring the *Alien / Prometheus* extended universe into the Stellaris sandbox.  
This project aims to unify lore from films, novels, comics, games, and official reference guides into a coherent and playable Stellaris experience.

## üìö Lore Coverage

This mod aims to incorporate lore from across the extended universe, including:

- **Alien (1979)**, **Aliens (1986)**, **Alien 3**, **Alien Resurrection**
- *Prometheus*, *Alien: Covenant*
- Colonial Marines Tech Manuals  
- Weyland-Yutani Reports  
- Novels & Comics (Titan Books, Dark Horse)  
- Games such as *Alien: Isolation* and *Fireteam Elite*

We treat contradictions as opportunities to build the most internally consistent version possible for Stellaris gameplay.
